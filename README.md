# roman-xmatch

Cross-correlate Nancy Grace Roman Space Telescope survey footprints with
pre-built astronomical catalogs using an interactive sky plot.

---

## Installation

```bash
pip install git+https://github.com/pmarcum/roman_xmatch.git
```

### Optional: HEALPix mask support

```bash
pip install roman-xmatch[healpix]
```

> **Windows users:** `tkinter` is not always included in Python distributions
> on Windows. If the GUI fails to launch, reinstall Python from python.org and
> check the "tcl/tk and IDLE" component during setup.

---

## Quick start

```bash
roman-xmatch
```

This opens the GUI. Select a footprint file, choose catalogs, and click **Run**.

## Catalogs

All catalogs ship pre-built with the package (small ones bundled in the wheel;
large ones downloaded on first use from GitHub Releases).

| Key | Description | Sources | Delivery |
|-----|-------------|--------:|---------|
| `abell` | Abell galaxy clusters (VizieR VII/110A) | ~2,700 | bundled |
| `mcxc2` | MCXC-II X-ray clusters (VizieR J/A+A/688/A187) | 2,221 | bundled |
| `xmm_clusters` | XMM X-CLASS DR1 clusters (VizieR J/MNRAS/423/1024) | 503 | bundled |
| `ngc_ugc` | NGC/IC + UGC galaxies (VizieR VII/118 + VII/26D) | ~26,000 | bundled |
| `chandra_gal` | Chandra CSC 2.1 X-ray galaxies | ~23,500 | bundled |
| `xmm_gal` | XMM 4XMM-DR13 X-ray galaxies (VizieR IX/69) | ~1,100 | bundled |
| `swift_gal` | Swift 2SXPS X-ray galaxies (VizieR IX/58) | ~620 | bundled |
| `sdss` | SDSS DR18 spectroscopic galaxies | ~3M | GitHub Releases |
| `2masx` | 2MASS Extended Source Catalog (VizieR VII/233) | ~1.6M | GitHub Releases |
| `hst_deep` | HST deep fields — CANDELS + GOODS | ~38,000 | GitHub Releases |
| `custom` | User-supplied FITS or CSV file | — | local |

GitHub Releases catalogs are downloaded automatically on first use and cached
locally. No manual setup required.

## GUI overview

1. **Footprint file** — select a Roman survey footprint (FITS, CSV, or
   HEALPix `.fits` mask).
2. **Catalogs** — check any combination of catalogs to include.
3. **Match mode**
   - **AND** — return only sources present in *all* selected catalogs
     (within the tolerance radius).
   - **OR** — return sources present in *any* selected catalog.
4. **Tolerance** — positional matching radius in arcseconds (used for both
     AND deduplication and OR label suppression).
5. **Run** — executes the cross-match and opens the interactive sky plot.

### Interactive plot

- **Pan / zoom** — standard matplotlib toolbar or scroll wheel.
- **Labels** — source labels appear automatically when sufficiently zoomed in;
  label density scales with zoom level per catalog.
- **Export** — saves the current view as a PNG or the matched source list as
  a CSV.

## Python API

```python
from roman_xmatch import pipeline

results = pipeline.run(
    footprint_file   = "my_survey.fits",
    catalogs         = ["ngc_ugc", "chandra_gal", "sdss"],
    match_mode       = "OR",
    tolerance_arcsec = 5.0,
)

for key, table in results.items():
    print(key, len(table), "sources")
```

## Rebuilding the catalog data files

The pre-built `.fits.gz` files are generated by `scripts/build_catalogs.py`.
You should not need to run this yourself — it is a developer tool for
regenerating the shipped data files when source catalogs are updated.

```bash
cd /path/to/roman_xmatch_repo
pip install astropy astroquery numpy pandas

# Build all catalogs (takes 1-2 hours; SDSS dominates)
python scripts/build_catalogs.py

# Build specific catalogs only
python scripts/build_catalogs.py --catalogs abell mcxc2 ngc_ugc

# Force rebuild even if files already exist
python scripts/build_catalogs.py --force
```

Large catalogs (sdss, 2masx, hst_deep) must be uploaded to GitHub Releases
after building — they are too large to bundle in the wheel.
